# 贝叶斯小组项目（Bayesian Group Project）
> 贝叶斯统计建模与对比分析


---

## 📋 项目简介

本项目为贝叶斯统计分析小组作业，旨在通过构建和比较不同的贝叶斯模型，探索模型选择、参数估计以及模型比较中的关键问题。项目包含完整的代码实现与数据分析流程。

---

## 🔍 核心研究问题

| 序号 | 问题 | 说明 |
|:---:|------|------|
| 1 | **为何两个模型结果一样？** | 探究两个不同贝叶斯模型在相同数据上产生一致结果的原因，分析模型等价性或参数可识别性问题 |
| 2 | **模型数量够不够？** | 评估当前模型集合是否充分覆盖假设空间，讨论模型比较的完备性与局限性 |
